\doxysubsection{Detailed Description}
Defines a 3-\/dimensional frustum.

A frustum in computer graphics is generally a volume of 3D space, defined as the part of a rectangular pyramid that lies between two planes perpendicular to its center line. A frustum is often used to represent what a \char`\"{}camera\char`\"{} sees in your 3D space.

The \mbox{\hyperlink{classrtech_1_1math_1_1_frustum}{Frustum}} class allows you to define a bounding frustum using a combined matrix that is generally the product of a view matrix and a projection matrix.

You can query a \mbox{\hyperlink{classrtech_1_1math_1_1_frustum}{Frustum}} object for any one of its bounding planes, for its corners, and for whether it intersects with a given object. Since objects that don\textquotesingle{}t intersect with your view frustum generally don\textquotesingle{}t need to be rendered, culling them quickly can save you a lot of rendering time.
